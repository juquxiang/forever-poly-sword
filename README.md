# forever-poly-sword
基于raylib开源视频游戏编程库做的小游戏
本项目环境为windows，下载后点击game.exe即可运行
build.bat是编译脚本，需要电脑有raylib环境并配置，具体配置可以参考 https://www.bilibili.com/video/BV13B4y1c7HX?p=1 里的环境搭建。
